# **App Name**: FrigoManager

## Core Features:

- Dashboard Module: Provides a general overview of the operation with key metrics and alerts.
- Bins and Materials Module: Manages the inventory and tracking of bins and other materials used in the fruit processing.
- Reception Module: Records the reception of fruit, including details about the exporter, producer, and quantity.
- Hydrocooler Module: Tracks the hydrocooling process, including temperature and duration for each batch of fruit.
- Cameras Module: Monitors the storage conditions in the cooling chambers, including temperature and humidity levels.
- Dispatch Module: Manages the dispatch of processed fruit, including details about the client, destination, and quantity.
- Master Data Management Module: Centralized module for managing exporters, producers, bins/materials, other clients, packagings, users, and profiles.

## Style Guidelines:

- Primary color: Light green (#A7D1AB), evoking freshness and natural produce.
- Background color: Very light green (#F0F4F1), almost white, providing a clean and fresh feel.
- Accent color: A slightly more saturated green (#74B72E), used for interactive elements and highlights to guide the user.
- Body and headline font: 'PT Sans', a humanist sans-serif for a modern and warm feel suitable for both headlines and body text.
- Simple, flat icons to represent each module and action, focusing on clarity and ease of understanding.
- Tabular layout for data display, with clear separation between columns and rows to improve readability. Use of modals or side panels for CRUD operations on master data.
- Subtle transitions and animations for loading data and displaying notifications, providing a smooth user experience.
